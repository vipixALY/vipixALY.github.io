Unità 4: Sistemi Operativi

1. Introduzione ai sistemi operativi

Il sistema operativo (OS) è il software fondamentale che agisce da intermediario tra l'hardware del computer e l'utente. Senza un sistema operativo, un computer non sarebbe in grado di eseguire applicazioni o gestire risorse come memoria, CPU e dispositivi di input/output. I sistemi operativi moderni sono progettati per essere intuitivi e facili da usare, ma dietro le quinte svolgono compiti complessi per garantire che tutto funzioni senza intoppi.

1.1 Funzioni principali

  • Gestione della memoria: Alloca e libera memoria RAM, gestisce la memoria virtuale e previene conflitti tra processi. La memoria virtuale consente al sistema di utilizzare spazio su disco come estensione della RAM, migliorando le prestazioni.
  • Gestione dei processi: Pianifica l'esecuzione dei programmi, gestisce il multitasking e assegna risorse della CPU. Il sistema operativo utilizza algoritmi di scheduling per decidere quale processo deve essere eseguito e per quanto tempo.
  • Gestione dei file: Organizza i dati in file e directory, gestisce permessi e operazioni di lettura/scrittura. I file system moderni supportano funzionalità avanzate come la crittografia e la compressione dei dati.
  • Gestione delle periferiche: Controlla dispositivi hardware tramite driver e gestisce input/output. I driver sono software specializzati che consentono al sistema operativo di comunicare con dispositivi specifici come stampanti, schede grafiche e dispositivi di archiviazione.
  • Sicurezza: Protegge il sistema da accessi non autorizzati e malware. I moderni sistemi operativi includono firewall integrati, crittografia dei dati e funzionalità di rilevamento delle intrusioni.

1.2 Tipi di sistemi operativi

  • Desktop OS: Windows, macOS, Linux (per utenti finali). Questi sistemi operativi sono progettati per essere facili da usare e supportano una vasta gamma di applicazioni.
  • Server OS: Windows Server, Linux Server (per gestione reti e servizi). I sistemi operativi server sono ottimizzati per gestire carichi di lavoro pesanti e supportano funzionalità come la virtualizzazione e il bilanciamento del carico.
  • Mobile OS: Android, iOS (per smartphone e tablet). Questi sistemi operativi sono progettati per dispositivi con risorse limitate e supportano applicazioni mobili e funzionalità touch.
  • Real-Time OS: Utilizzati in sistemi embedded (es. automazione industriale). Questi sistemi operativi sono progettati per eseguire compiti in tempo reale con precisione millisecondaria.

2. Microsoft Windows

Windows è il sistema operativo più diffuso al mondo, con una quota di mercato superiore al 75% sui PC desktop. La sua popolarità è dovuta alla facilità d'uso, alla vasta compatibilità hardware/software e al supporto continuo da parte di Microsoft. Windows è utilizzato in una varietà di ambienti, dalle case agli uffici, dalle scuole alle grandi aziende.

2.1 Storia e versioni principali

  • Windows 95: Introdusse il menu Start e il concetto di plug-and-play. Questa versione rivoluzionò l'uso dei PC, rendendoli accessibili a un pubblico più ampio.
  • Windows XP: Stabilità e interfaccia user-friendly (2001-2014). Windows XP è stato uno dei sistemi operativi più longevi e amati di sempre.
  • Windows 7: Miglioramenti prestazionali e sicurezza (2009). Windows 7 è stato acclamato per la sua stabilità e facilità d'uso.
  • Windows 10: Integrazione con cloud e aggiornamenti continui (2015). Windows 10 ha introdotto funzionalità come Cortana e il Microsoft Store.
  • Windows 11: Interfaccia modernizzata e supporto per app Android (2021). Windows 11 offre un design più moderno e miglioramenti nelle prestazioni.

2.2 Architettura

Basato su kernel ibrido (monolitico e microkernel), supporta:

  • File system NTFS, exFAT, FAT32. NTFS offre funzionalità avanzate come la crittografia e i permessi di file.
  • API Win32 e .NET Framework. Queste API consentono agli sviluppatori di creare applicazioni potenti e integrate.
  • Virtualizzazione tramite Hyper-V. Hyper-V consente di eseguire più sistemi operativi su un singolo hardware.

3. L'interfaccia utente di Windows 10

Windows 10 combina elementi tradizionali con funzionalità moderne, offrendo un'esperienza utente versatile. L'interfaccia è progettata per essere intuitiva e personalizzabile, consentendo agli utenti di adattarla alle proprie esigenze.

3.1 Componenti chiave

  • Start Menu: Accesso a programmi, file recenti e ricerca globale. Il menu Start è stato riprogettato per essere più funzionale e personalizzabile.
  • Taskbar: Collegamenti rapidi, anteprime applicazioni e notifiche. La taskbar è uno strumento essenziale per la navigazione e la gestione delle applicazioni.
  • Action Center: Notifiche e impostazioni rapide (Wi-Fi, Bluetooth, modalità notte). L'Action Center è un hub centrale per le notifiche e le impostazioni rapide.
  • Virtual Desktops: Creazione di più desktop per organizzare il lavoro. I desktop virtuali consentono di separare le attività in spazi di lavoro distinti.

3.2 Personalizzazione avanzata

  • Temi: Cambio colori, sfondi e suoni. I temi consentono di personalizzare l'aspetto del sistema operativo.
  • Impostazioni avanzate: Regolazione risoluzione schermo, gestione dispositivi, opzioni di alimentazione. Le impostazioni avanzate offrono un controllo completo sul sistema.
  • Cortana: Assistente vocale integrato per ricerche e comandi vocali. Cortana può aiutare a trovare file, impostare promemoria e molto altro.

4. I file e il file system

I file sono l'unità base di archiviazione dei dati, organizzati in directory (cartelle) e gestiti dal file system. Il file system è responsabile dell'organizzazione, dell'archiviazione e del recupero dei dati su dispositivi di archiviazione come dischi rigidi e SSD.

4.1 Struttura dei file

  • Nome ed estensione: Esempio: documento.docx (.docx indica un file Word).
  • Percorsi: Esempio: C:\Utenti\NomeUtente\Documenti\file.txt.
  • Attributi: Sola lettura, nascosto, sistema.

4.2 Operazioni comuni

  • Copia/Sposta: CTRL+C / CTRL+V o drag-and-drop.
  • Compressione: Creazione file ZIP per risparmiare spazio.
  • Ricerca: Utilizzo della barra di ricerca in Esplora File.

5. Gestione delle attività

Il Task Manager è uno strumento essenziale per monitorare e gestire le risorse del sistema. Consente agli utenti di visualizzare i processi in esecuzione, monitorare l'utilizzo delle risorse e terminare i processi non rispondenti.

5.1 Funzionalità principali

  • Processi: Visualizza CPU, memoria, disco e utilizzo rete per ogni applicazione.
  • Prestazioni: Grafici in tempo reale per CPU, RAM, disco e GPU.
  • Avvio: Gestione programmi che partono all'avvio del sistema.

5.2 Ottimizzazione

  • Chiusura programmi non rispondenti.
  • Identificazione processi ad alto utilizzo di risorse.
  • Disabilitazione programmi superflui all'avvio.

6. Linux e l'open source

Linux è un sistema operativo basato su kernel open source, ampiamente utilizzato in server, supercomputer e dispositivi IoT. La sua flessibilità e sicurezza lo rendono una scelta popolare per sviluppatori e amministratori di sistema.

6.1 Caratteristiche principali

  • Licenza GPL: Codice sorgente liberamente modificabile e distribuibile.
  • Terminale: Potente interfaccia a riga di comando (CLI).
  • Package manager: Strumenti come APT (Debian/Ubuntu) o YUM (Fedora) per installare software.

6.2 Distribuzioni popolari

  • Ubuntu: Ideale per principianti, ampia comunità.
  • Fedora: Innovativo, sponsorizzato da Red Hat.
  • Debian: Stabile, base per molte distribuzioni.

7. L'interfaccia utente di Ubuntu

Ubuntu utilizza l'ambiente desktop GNOME, noto per la sua semplicità e modernità. L'interfaccia è progettata per essere intuitiva e facile da usare, con un focus sulla produttività.

7.1 Componenti principali

  • Dash: Barra laterale per lanciare applicazioni.
  • Workspaces: Creazione di più aree di lavoro per organizzare le finestre.
  • Software Center: Installazione di applicazioni da repository ufficiali.

7.2 Personalizzazione

  • Installazione temi e icone tramite GNOME Tweaks.
  • Creazione di shortcut personalizzati.
  • Gestione estensioni per aggiungere funzionalità.

8. Confronto tra Windows e Linux

Ecco un confronto dettagliato tra i due sistemi operativi:

Criterio Windows Linux
Costo Licenza a pagamento (tranne Windows 10/11 in modalità S) Completamente gratuito
Sicurezza Maggiore esposizione a malware Architettura più sicura, meno vulnerabile
Customizzazione Limitata Illimitata (kernel modificabile)
Supporto hardware Ottimale (driver per quasi tutti i dispositivi) Variabile (dipende dalla distribuzione)
Software disponibile Vasta gamma di applicazioni commerciali Open source e software libero

9. Macchine virtuali

Le macchine virtuali (VM) permettono di eseguire più sistemi operativi su un singolo hardware. Sono utilizzate per scopi di testing, sviluppo e isolamento delle applicazioni.

9.1 Vantaggi

  • Isolamento: Ogni VM è indipendente dalle altre.
  • Flessibilità: Esecuzione di OS diversi (es. Windows su Linux).
  • Backup: Facile creazione di snapshot per ripristino.

9.2 Software popolari

  • VirtualBox: Gratuito e open source.
  • VMware Workstation: Professionale, a pagamento.
  • Hyper-V: Integrato in Windows Pro/Enterprise.

10. Programmi di utilità

I plugin sono strumenti essenziali per mantenere il tuo sistema efficiente e sicuro. Questi strumenti aiutano a gestire le risorse, migliorare le prestazioni e proteggere i dati.

10.1 Strumenti di sistema

  • Deframmentazione disco: Ottimizza l'archiviazione su HDD.
  • Pulizia disco: Rimuove file temporanei e non necessari.
  • Backup e ripristino: Crea copie di sicurezza dei dati.

10.2 Utility third-party

  • Antivirus: Protezione da malware (es. Avast, Norton).
  • Ottimizzazione: Migliora prestazioni (es. CCleaner).
  • Recupero dati: Ripristina file cancellati (es. Recuva).

Conclusioni

I sistemi operativi sono il cuore di ogni computer, e la scelta tra Windows, Linux o altri dipende dalle esigenze specifiche. Windows offre facilità d'uso e compatibilità, mentre Linux garantisce controllo e sicurezza. Strumenti come le macchine virtuali e i programmi di utilità estendono le possibilità operative, rendendo i sistemi operativi strumenti potenti e versatili per ogni tipo di utente.
